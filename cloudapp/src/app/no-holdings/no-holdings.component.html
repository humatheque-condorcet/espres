Bienvenue sur l'app Humacheck !

Cette application vous permet de contrôler :

  • La validité du format d'une cote lot ou corpus ;
  • La cohérence entre cote et localisation ;
  • La validité du lot ou du corpus choisi dans Plac.

Elle vous permet également de modifier la localisation et la cote en appliquant ces contrôles.

Attention ! Cette application ne fonctionne que pour les cotes monographies de LA dans les localisations définitives.

 

Attention ! Cette application est utilisable sur les écrans de holdings ou dans l'éditeur de métadonnées pour une holding !
